# TheEngine - A Lightweight Data Oriented Game Engine (Working Title)

[![Windows Build](https://github.com/aka411/the-engine/actions/workflows/windows-build.yml/badge.svg)](https://github.com/aka411/the-engine/actions/workflows/windows-build.yml)

TheEngine is a lightweight, modular game engine being built from scratch with a focus on Data-Oriented Design. It's currently in its early stages, and the name "TheEngine" is a working title that may change as the project evolves.
